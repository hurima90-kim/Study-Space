## 🗂 구현 목록

`Task생성, 수정, 삭제`

- 내용 작성 후 task 추가
- 연필 아이콘 클릭 후 task 수정
- 휴지통 아이콘을 클릭 시 해당id에 맞는 Task가 삭제

`Task 필터링`

- 설정된 Task의 상태(All, Active, Complete)에 따른 필터링
- 각 상태에 맞는 task 갯수 출력

`전역상태관리 및 비동기처리`

- React-redux를 활용한 전역상태관리
- Redux-saga를 활용한 비동기처리
