# Note-Taking
A simple Note-Taking app built using JavaScript

## 주요기능
1. 웹상에서 기억해야 할 내용을 저장 및 삭제, 읽기가 가능합니다.
2. 작성후 저장한 메모는 모달창으로 작성한 내용을 다시 확인가능합니다.
3. 작성된 메모들은 사용자의 로컬스토리지에 저장됩니다.
4. 로컬스토리지에 저장된 메모들은 삭제시에 로컬스토리지에서도 삭제됩니다.
